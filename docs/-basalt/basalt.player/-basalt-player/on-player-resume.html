onPlayerResume

fun onPlayerResume(player: AudioPlayer): Unit

Fired by Lavaplayer when the AudioPlayer is resumed.

This sends a PLAYER_PAUSE event, wrapped in a Dispatch Response, across a WebSocketChannel. Note: This sends the same event as when the player is paused, but the data is set to false (for "is paused").

Parameters

player - The AudioPlayer object itself.
