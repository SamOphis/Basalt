onTrackStart

fun onTrackStart(player: AudioPlayer, track: AudioTrack): Unit

Fired by Lavaplayer when an AudioTrack starts.

This sends a TRACK_STARTED event, wrapped in a Dispatch Response, across a WebSocketChannel. Additionally, it starts an Update Task which sends PLAYER_UPDATE events that are NOT wrapped in a Dispatch Response every 5 seconds.

Parameters

player - The AudioPlayer object itself.

track - the AudioTrack that was started.
