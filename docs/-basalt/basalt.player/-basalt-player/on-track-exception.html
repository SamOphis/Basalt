onTrackException

fun onTrackException(player: AudioPlayer, track: AudioTrack, exception: FriendlyException): Unit

Fired by Lavaplayer when an exception was thrown during playback of an AudioTrack.

This sends a TRACK_EXCEPTION event, wrapped in a Dispatch Response, across a WebSocketChannel.

Parameters

player - The AudioPlayer object itself.

track - The AudioTrack that was playing.

exception - The exception that was thrown.
