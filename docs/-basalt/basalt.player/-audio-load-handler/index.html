AudioLoadHandler

class AudioLoadHandler : AudioLoadResultHandler

Class which loads sources based on provided identifiers asynchronously.

Each instance of this class can only be used once, and an exception will be thrown as well as an error log being produced each time this is used more than once -- which it is not, if using the normal, stable Basalt release.
