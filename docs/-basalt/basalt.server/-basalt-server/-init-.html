BasaltServer()

Server instance which is the starting point for every other part of Basalt.

This class is only meant to run/started up once, although it technically allows multiple start ups. From this class, the Undertow WebSocket is created and the configuration options are parsed and applied.

The Jsoniter options are also set upon this server being started, as well as Magma being setup too.
