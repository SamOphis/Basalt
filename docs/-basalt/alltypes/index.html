The main Basalt Module itself, which contains everything inside the project.

All Types

basalt.player.AudioLoadHandler

Class which loads sources based on provided identifiers asynchronously.
basalt.server.AudioSender

Sender class which takes in Opus frames from Lavaplayer and passes them to Magma, where they are then subsequently sent to Discord.
basalt.util.AudioTrackUtil

The utility class which converts AudioTracks to easily-transportable data and vice versa.
basalt.player.BasaltPlayer

A player class associated with a SocketContext and a Guild ID, with a "lifecycle" determined by incoming requests.
basalt.server.BasaltServer

Server instance which is the starting point for every other part of Basalt.
basalt.server.ErrorResponses

An enum of responses that are returned in Dispatched ERROR Responses, as they're written here.
basalt.player.LoadResult

A result class which represents a singular, successful load result.
basalt.player.PlayerMap

Type alias equivalent to an Object2ObjectOpenHashMap<String, BasaltPlayer>
basalt.player.ResultStatus

Enum which corresponds to how a source loaded.
basalt.player.SocketContext

Context class which stores a reference to a BasaltServer, a WebSocketChannel and the User ID associated with that channel, as well as a PlayerMap and a Sequence Number of Sent, Successful Events.
basalt.server.SocketContextMap

Type alias which is equal to Object2ObjectOpenHashMap<WebSocketChannel, SocketContext>
basalt.server.WebSocketListener

The listener class which responds to WebSocket Events, including (but not limited to) incoming messages.
